My Projects

HMI Speed Sensor - This Human Machine Interface (HMI) was created using Microsoft Visual Basic software, which was implemented to control and monitor the speed of a DC motor that is integrated with the Rotary Encoder and connected to an Arduino microcontroller using serial communication.

Board Controller and Monitoring for Aquarium - This project is one of the products created by the team while interning at CV. Elmech Technology. The purpose of this project is to monitor and control the water quality in ornamental fish aquariums or indoor fish hatchery ponds. Ammonia (NH3), pH, and temperature levels are the parameters that are monitored and controlled. Monitoring data will be collected and stored in a database before being displayed on the web page and mobile app.

Prototype of Bridge Structure Monitoring System Using LoRa Technology - This project is the result of the final project team. The system in this project consists of two nodes, namely the end-node transmitter and the end-node receiver. The main data on this system is acceleration data read by the AKF393B sensor and acquired by the ESP32 microcontroller using modbus communication on the transmitter's end-node, then the data is sent to the receiver's end-node using LoRa technology. The data received is processed by a certain method so that it becomes frequency data, which is a parameter for testing the bridge service level. Some documentaion of the project's final results is not displayed due to intellectual property issues.
